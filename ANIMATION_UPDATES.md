# 動畫與互動優化更新

## ✨ 新增功能

### 1. 載入動畫
- 頁面載入時顯示 "LOADING..." 動畫
- 淡入淡出效果，提升專業感

### 2. 滾動進度條
- 頁面頂部顯示滾動進度
- 漸層色彩（暖色調）
- 即時反映閱讀進度

### 3. 視差效果增強
- Hero 區背景隨滾動移動
- Hero 內容在滾動時淡出並移動
- 標題支援滑鼠移動視差（data-parallax 屬性）

### 4. 打字機效果
- About 區文字逐字顯現
- 當元素進入視野時自動觸發
- 速度：30ms/字元

### 5. 社群媒體圖示動畫
- 浮動動畫（上下移動）
- 每個圖示延遲不同，創造波浪效果
- Hover 時放大並加陰影

### 6. 標題裝飾線
- Hero 標題下方有動畫線條
- Section 標題下方有裝飾細線
- 增加視覺層次

### 7. 圖片懸停效果升級
- **About 照片**：
  - 放大並輕微旋轉
  - 外框線條擴大動畫
  - 加強陰影效果

- **Gallery 圖片**：
  - 放大並旋轉
  - 顯示 🔍 圖示覆蓋層
  - 邊框動畫

### 8. 音樂平台圖示
- Zoom-in 進場動畫（分批出現）
- Hover 時脈衝跳動效果
- 平台專屬顏色（Spotify 綠、YouTube 紅等）
- 加強陰影效果

### 9. 滑鼠游標效果增強
- Hover 互動元素時游標放大
- 更流暢的跟隨動畫
- 適用於連結、按鈕、圖片

### 10. 文字淡入動畫
- 副標題和文字內容從下方滑入
- IntersectionObserver 觸發
- 避免一次載入全部內容

## 🔧 技術細節

### CSS 動畫
- 使用 `@keyframes` 定義關鍵幀
- `transition` 屬性控制過渡
- `transform` 實現位移、縮放、旋轉

### JavaScript 互動
- `IntersectionObserver` 監測元素進入視野
- `requestAnimationFrame` 優化性能
- 事件委託處理大量互動元素

### AOS 設定
```javascript
AOS.init({
    duration: 1000,           // 動畫持續時間
    easing: 'ease-out-cubic', // 緩動函數
    once: true,               // 只播放一次
    offset: 100,              // 觸發偏移量
    disable: 'mobile'         // 手機停用（性能考量）
});
```

## 📱 響應式考量

- 手機裝置停用部分動畫（性能）
- 自訂游標在觸控裝置上隱藏
- 視差效果在小螢幕上簡化

## ⚡ 性能優化

1. **requestAnimationFrame** 用於滾動動畫
2. **節流（throttling）** 避免過度觸發
3. **IntersectionObserver** 取代滾動事件
4. **CSS transform** 優於 position 屬性
5. **will-change** 提示瀏覽器優化

## 🎯 使用建議

1. 所有動畫都已自動啟用
2. 如需停用某個效果，可註解對應的 JS 函數
3. 調整動畫速度：修改 CSS `transition` 或 JS `duration` 參數
4. 自訂顏色：修改 CSS variables

## 🐛 已知問題

- 部分舊版瀏覽器可能不支援某些效果
- 低性能裝置建議停用視差效果
- Safari 的 backdrop-filter 支援可能有限
